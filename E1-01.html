Exam 1 - 01 - Basics of Counting

1. Overall Learnings / Summary of the Topic

Combinatorics
The study of arrangements of objects. It essentially talks about the patterns that may arise when objects are randomly arranged with certain properties.
These include permutation, enumeration, and combinations of the set of given elements.

Basic Counting Principles

Product rule

 - n1(first task) * n2(second task) ways to do a procedure
- Applicable when a procedure is made up of separate tasks

When order matters:
Permutation is used
- n! / (n-r)!
- P(n,r) when the order of a choice matters (Also known as an r-permutation)

Set Theoretic Version of Product Rule
| AxB | = | A | * | B |
When tasks are disjoint

Sum Rule
If a task can be done with either one in n1 ways or n2 ways which no n-ways are the same:
n1 + n2 ways to do the task

Set Theoretic Version of the Sum Rule
| A U B | = | A | + | B |
When the tasks are disjoint

2. Analysis

The basics of counting refer to the primary and fundamental concepts of finding all the possible arrangements of a certain object or finding all the possible outcomes of an event that is likely to happen. This part talks about the main principle of Combinatorics which finding all the possibilities using the product and sum rules. 3. Applications, Real-Life Scenarios or Examples

Used in:
- Password Generation
- Voting Ballot
- Plate numbers
- Communication Networks
- Cryptography
- Computer Algorithms
- Probability for Machine Learning

Password Generation

Example 1.1.1 - Product Rule: if you have a choice of 26 letters (n = 26) and need to create a 4-letter password (m = 4), the product rule tells us that there are 26^4 = 456,976 possible combinations.

Example 1.1.2 - Sum Rule: In an election, if there are 'n1' candidates running for president and 'n2' candidates running for vice president, the sum rule tells us that there are 'n1 + n2' possible combinations of the president and vice president. This principle applies when choices are mutually exclusive.

4. Reflection and Thoughts

The Basics of Counting is the first and essential topic that is introduced in this subject. It is the foundation of most of the mathematical logic for computers. Product and sum rules on their own are mostly easy to understand in problem-solving. Combining these rules will have unique questions and results which I find a bit difficult to understand. This topic and the questions and problems along with it are difficult to distinguish from another topic in the next few lessons of the course, mainly permutations, and combinations. Overall, this topic was quite challenging as a first topic for this course but is important for the entirety of it.
